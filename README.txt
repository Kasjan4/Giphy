HOW TO RUN THE APP

1. To run the app from the terminal, navigate to the folder and first run 'npm i' to install the node modules.

2. Then run 'npm run app' to start the development server. Your default browser should open and you local can work on your local copy of the website.

3. Use 'npm run build' to build a new version of the code once you are happy with the end product, the source files will compile into a 'bundle.js' file in the root directory. This is the file browsers will use to read your React application.
